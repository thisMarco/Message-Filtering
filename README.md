# Message-Filtering
Napier University [SET09102 2019]

**Scenario and Requirement**

Euston is a medium-sized city. Euston Leisure is an association of all the sport centres in the city.
You are required to develop a service, namely Euston Leisure Messaging (ELM), which will validate, sanitize and categorise incoming messages
to Euston Leisure in the form of SMS text messages, emails and Tweets.
